# 產品定位與雙軌定價指南 (Product Strategy & Dual-Track Pricing)

這份文件記錄了「LINE AI 店長服務」的核心商業模式與價格體系。在任何後續開發中，都必須遵循此雙軌架構，確保前端動線清晰，且客戶不混淆。

## 📍 核心商業板塊 (The Dual-Track Business Model)

本專案分為兩大完全獨立的業務線：

### Track A: 直客零售版 (B2C / B2B Retail) —— 主戰場
- **對象：** 一般店家（例：單間瑜珈館）、連鎖總部（例：八方雲集）。
- **痛點：** 不懂程式、不需 API Key、想要開箱即用的「管理介面」來經營官方帳號。
- **介面：** 購買後進入「AICommandConsole」（黑金屬風 Dashboard），可看報表、下達防禦指令。
- **入口位置：** 網站首頁主定價區。

### Track B: 系統商批發版 (SaaS to SaaS) —— 隱藏戰場
- **對象：** POS 系統開發商、網頁設計接案公司、LINE CRM 廠商。
- **痛點：** 自己會寫程式（有工程團隊），但不想從頭訓練 AI。只想買我們的「AI大腦 API」裝進他們原本的軟體裡賣給別人。
- **介面：** 購買後進入「Partner Portal」，取得 `Partner Token`，查看 API 文件並分配席位 (Slots)。
- **入口位置：** 首頁最底部的低調按鈕 (`/saas-partnership` 落地頁)。

---

## 💰 服務項目與價格總表 (Pricing Tiers)

### Track A: 直客零售版 (顯示於首頁)

1. **Lite 輕量版 (單店)**
   - **定價：** **NT$ 399 / 月**
   - **功能定位：** 語言工作者（聊天、客服、產品介紹）。
   - **規格：** 每月 5,000 則對話，免自備 API Key。
   - **目標：** 讓小型店家 24 小時接單，不再漏接客戶。

2. **Pro 專業版 (單店) 🌟 主推**
   - **定價：** **NT$ 990 / 月**
   - **功能定位：** 經營管家（涵蓋庫存查詢、算毛利、訂單追蹤分析）。
   - **規格：** 每月 20,000 則對話，免自備 API Key，全方位顧店。
   - **目標：** 節省老闆時間，提升生產力的最佳選擇。

3. **Max 衝刺版 (單店 / 高層級需求)**
   - **定價：** **NT$ 2,490 / 月**
   - **功能定位：** 行銷大砲，無限制擴展版。
   - **規格：** 功能最全，極高用量對話，可選擇「是否帶自己的 API Key」。
   - **目標：** 超大流量電商名店或需高運算量的小組。

4. **HQ 連鎖 / 加盟總部方案 (多店管理) 👑**
   - **定價：** **專案報價 (或依總分店數階梯計費)**
   - **功能定位：** 總部戰情室與一鍵同步大腦。
   - **規格：** 總公司統一部署 Master Prompt 送達全台 50 家分店，能看全品牌總數據與分店營收數據，並設定各分店 AI 權限。
   - **目標：** 連鎖餐飲、大型健身房。

---

### Track B: 系統商批發版 (隱藏入口 `/saas-partnership`)

1. **試水溫選**
   - **定價：** **NT$ 5,500 / 月**
   - **功能：** 獲得 20 個機器人席位 (Slots) 的 API 權限，用來開發測試或是少量派給客戶。

2. **成長方案 (主推)**
   - **定價：** **NT$ 16,000 / 月**
   - **功能：** 獲得 50 個機器人席位，含所有產業知識庫板模存取權。

3. **企業授權**
   - **定價：** **專案報價**
   - **功能：** 無限席位，可要求客製化演算法或完全地端授權部署。
